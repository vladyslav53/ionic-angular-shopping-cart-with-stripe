
### Ionic 5 shopping cart using stripe payment 

Clone and run npm install

## Install the Strip cordova plugin 

```
ionic cordova plugin add cordova-plugin-stripe
npm install --save @ionic-native/stripe@4
```

## Stripe (Test) Key
